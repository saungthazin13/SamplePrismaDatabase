# SamplePrismaDatabase
1.Clone & enter
  git clone https://github.com/saungthazin13/SamplePrismaDatabase.git
  cd SamplePrismaDatabase

2.Install dependencies
  npm install

3.Database setup
.envမှာပြင်ရန်

4.Run migrations
  npx prisma migrate dev --name init

5.Launch Prisma Studio
  npx prisma studio
